# Common Network Attacks

## 1. Man-in-the-Middle (MITM)

### Description

An attacker intercepts communication between two parties.

### Risks

* Data theft
* Credential exposure
* Privacy violations

### Prevention

* Use HTTPS
* Use VPNs
* Enable encryption

---

## 2. Distributed Denial of Service (DDoS)

### Description

Attackers overwhelm a server with excessive traffic.

### Risks

* Service outages
* Business disruption

### Prevention

* Traffic filtering
* Rate limiting
* DDoS protection services

---

## 3. Packet Sniffing

### Description

Capturing network packets to analyze transmitted data.

### Risks

* Information disclosure
* Credential theft on insecure networks

### Prevention

* Use encrypted protocols
* Secure Wi-Fi networks
* VPN usage
