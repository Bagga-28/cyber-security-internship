# Network Security Overview

## What is Network Security?

Network security is the practice of protecting computer networks and connected devices from unauthorized access, misuse, and cyber threats.

## Key Concepts

### IP Address

A unique address assigned to devices connected to a network.

### DNS (Domain Name System)

Translates domain names into IP addresses.

### HTTP

A protocol used for transferring web pages.

### HTTPS

A secure version of HTTP that uses encryption.

## Importance of Network Security

* Protects sensitive information
* Prevents unauthorized access
* Ensures system availability
* Maintains data integrity
