# Understanding Network Security & Basic Traffic Monitoring

## Objective

The purpose of this task is to understand how data travels across networks, identify common network threats, and learn the basics of network traffic monitoring using safe and legal methods.

## Topics Covered

* Network Security Fundamentals
* IP, DNS, HTTP, and HTTPS
* Common Network Attacks
* Traffic Monitoring
* Packet Analysis

## Tools and Resources

* Networking Basics Tutorials
* Wireshark Beginner Guides
* Network Security Articles

## Learning Outcomes

* Understanding how networks communicate
* Recognizing common network threats
* Learning the importance of traffic monitoring
* Developing cybersecurity awareness

## Ethical Statement

All activities were conducted for educational purposes only. No unauthorized network monitoring or attacks were performed.
