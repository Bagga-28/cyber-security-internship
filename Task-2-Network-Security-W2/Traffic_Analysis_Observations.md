# Traffic Analysis Observations

## Objective

To understand how network traffic is transmitted and identify common protocols.

## Observations

### DNS Traffic

Observed DNS requests translating domain names into IP addresses.

### HTTP/HTTPS Traffic

Observed web traffic generated while browsing websites.

### TCP Communication

Observed connection establishment and data transmission using TCP.

## Learning Outcome

Traffic analysis helps security professionals understand network behavior, troubleshoot connectivity issues, and identify suspicious activity.
