What is Cyber Security?
Cyber security is the practice of protecting systems, networks, and data from cyber attacks.

Common Cyber Threats
- Phishing
- Malware
- Ransomware
- Social Engineering
- Brute Force Attacks

Digital Safety Best Practices
- Use strong passwords
- Enable Two-Factor Authentication (2FA)
- Keep software updated
- Avoid suspicious links
- Back up important data

Conclusion
Cyber security awareness helps users stay safe online and protect sensitive information.
