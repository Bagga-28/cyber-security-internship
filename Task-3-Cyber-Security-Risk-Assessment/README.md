Scenario
A small company stores employee and customer data online.

Assets
- Employee Records
- Customer Data
- Company Website
- Email Accounts

Threats
- Phishing Attacks
- Malware
- Data Breaches
- Weak Passwords

Vulnerabilities
- No Multi-Factor Authentication
- Weak Password Policies
- Unpatched Software

Recommendations
- Enable MFA
- Use Strong Passwords
- Regular Security Updates
- Employee Security Awareness Training

Conclusion
Proper security controls can significantly reduce cyber security risks.
