# Cyber Threat Intelligence (CTI) Gathering and Analysis

## Objective

The objective of this task is to understand Cyber Threat Intelligence (CTI), how threat information is collected, analyzed, and used to improve an organization's security posture.

---

# What is Cyber Threat Intelligence?

Cyber Threat Intelligence (CTI) is information about cyber threats, threat actors, attack techniques, and vulnerabilities that helps organizations make informed security decisions.

## Benefits of CTI

* Early threat detection
* Better incident response
* Improved risk management
* Enhanced security awareness

---

# Types of Threat Intelligence

## 1. Strategic Intelligence

High-level information focused on business risks and long-term trends.

### Audience

* Executives
* Management

---

## 2. Tactical Intelligence

Information about attacker tactics, techniques, and procedures (TTPs).

### Audience

* Security Analysts
* SOC Teams

---

## 3. Operational Intelligence

Information about specific attacks and campaigns.

### Audience

* Incident Response Teams

---

## 4. Technical Intelligence

Technical indicators used to identify threats.

### Examples

* IP Addresses
* Domains
* File Hashes
* URLs

---

# Indicators of Compromise (IOCs)

## Sample Indicators

### IP Addresses

Suspicious external IP addresses communicating with infected systems.

### Domains

Malicious domains used for command-and-control communications.

### File Hashes

SHA-256 hashes used to identify known malicious files.

### URLs

Malicious URLs used in phishing campaigns.

# Recommendations

## User Security

* Conduct phishing awareness training.
* Use strong passwords.
* Enable Multi-Factor Authentication (MFA).

## System Security

* Apply security patches regularly.
* Maintain endpoint protection.
* Restrict administrative privileges.

## Monitoring

* Monitor logs continuously.
* Use SIEM solutions.
* Investigate suspicious alerts promptly.

# Threat Intelligence Lifecycle

1. Planning
2. Collection
3. Processing
4. Analysis
5. Dissemination
6. Feedback

This lifecycle ensures threat information is collected and transformed into actionable intelligence.

Cyber Threat Intelligence helps organizations identify, analyze, and respond to cyber threats proactively. By gathering intelligence from multiple sources and analyzing indicators of compromise, organizations can strengthen their security posture and improve incident response capabilities.
